== Description ==

This is Bootstrap Flex Slider plugin. Its used for Bootstrap v3x Themes. Its have admin controls options, Navigation and Directions options in admin panel

= Custom Options =
	
	1. Install Plugin
	2. Add Short code [display_flexslider]
